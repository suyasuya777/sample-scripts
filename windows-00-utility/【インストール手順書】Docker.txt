■Docker

１．Docker Desktop のインストール

　1-1.以下のＵＲＬにアクセス
　　https://www.docker.com/products/docker-desktop/

　1-2.インストーラをダウンロード
　　Download Docker Desktop

　1-3.インストーラを実行
　　インストール中のオプションで
　　「Use WSL 2 instead of Hyper-V」をチェックする

　1-4.インストール完了後、再起動を求められたら必ず再起動

　1-5.Dockerのインストール確認
　　docker --version

　　スタートメニュー → Docker Desktop
　　左下が “Engine running” になれば成功
